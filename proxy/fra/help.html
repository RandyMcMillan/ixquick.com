Recherche Internet par Ixquick
www.ixquick.com ProxyMis en surbrillanceixquick.com permet d’obtenir des résultats de recherche exacts dans des conditions de protection complète des informations personnelles de votre vie privée.
  • Lorsque vous cliquez directement sur un résultat de recherche, vous quittez la protection d’Ixquick.
  • Une connexion directe avec le site Internet tiers est établie.
  • Le site Internet peut voir et enregistrer votre adresse IP, placer des fichiers témoins (cookies) sur votre navigateur, et éventuellement lire les cookies mis en place par d’autres.
  • Le site Internet peut observer et enregistrer votre comportement, en particulier les liens sur lesquels vous cliquez et les pages que vous affichez.
  • L’avantage c’est que les pages se téléchargent plus rapidement, puisque vous affichez directement le contenu à partir de sa source.
  • L’option Sans Proxyse justifie si vous n’êtes pas concerné par la protection de votre vie privée, ou si vous avez confiance dans le site Internet, ou lorsque vous souhaitez être identifié, par exemple quand vous vous connectez à votre compte bancaire.

Recherche Internet par Ixquick
www.ixquick.comProxyMis en surbrillanceixquick.com permet d’obtenir des résultats de recherche exacts dans des conditions de protection complète des informations personnelles de votre vie privée.
  • Ixquick va sur le site Internet de votre choix (que vous avez sélectionné), récupére la page et l’affiche pour vous.
  • Vous êtes invisible pour le site Internet. Il ne voit que l’adresse IP d’Ixquick, pas la vôtre.
  • Comme vous n’êtes jamais en contact direct avec le site Internet,personne ne peut voir ou conserver des cookies sur votre navigateur.
  • A partir du site Internet vous pouvez cliquer sur les pages où figurent des liens et Ixquick proxy les affichera pour vous.
  • Pendant que vous utilisez le serveur proxy votre vie privée est sous la protection permanente de l’exceptionnelle politique de confidentialité d’Ixquick. (Pas d’enregistrement de l’adresse IP, pas d’utilisation de cookies d’identification, pas de conservation des activités de recherche ou de navigation)
  • Le service Ixquick Proxy offre une complète protection de votre vie privée et un anonymat total.
  • Le désavantage c’est que les pages se téléchargent moins vite puisque Ixquick doit rechercher leurs contenus et les afficher une nouvelle fois.
Remarques et Restrictions supplémentaires :
  • Pour des raisons de sécurité, il n’est pas possible d’utiliser le serveur proxy pour remplir des formulaires avec une saisie de texte.
  • Le serveur proxy supprime le code « javascript » des sites consultés, de sorte que certaines fonctionnalités peuvent ne pas être opêrationnelles.
  • Le serveur proxy ne téléchargera pas les „cadres“ appartenant à un domaine externe différent de celui de l’auteur de la référence.
  • Pour l'instant le service proxy ne s'applique pas encore aux recherches d'image ou de video dans leur totalite.
  • Pour encore davantage de protection, vous pouvez désactiver les scripts depuis ixquick-proxy.com avec le plugin 'NoScript' de Firefox.

En supprimant une bonne partie du code „javascript“ des pages sous proxy, Ixquick Proxy évite la plupart des problèmes de sécurité.

Pour une sécurisation supplémentaire, Ixquick recommande de désactiver Java Script, Java Flash et tous les autres scripts de l’ensemble du domaine ixquick-proxy.com et de ses sous-domaines.

Pour cela:
1. Vérifiez bien que vous utilisez le navigateur Firefox
2. Installez le module d’extension „No script“ (No script Add-on)

3. Pour une plus grande facilité d’utilisation, configurez No script afin
  A. d’autoriser les Scripts globalement (dans leur totalité) et
  B. de désactiver les Scripts de Ixquick-proxy.com et de ses sous-domaines

Ou, pour être protégé même lorsque vous n’utilisez pas Ixquick proxy, mais cela avec certains inconvénients, configurez No Script afin:
  A. de bloquer les Scripts dans leur totalité et
  B. d’activer les Scripts d’ixquick.com, de ses sous-domaines et de tous les autres sites web fiables, mais pas ixquick-proxy.com.
