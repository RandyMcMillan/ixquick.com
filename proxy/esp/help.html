Ixquick Web Search
www.ixquick.com ProxyDestacar¡Ixquick.com ofrece resultados de búsqueda precisos con total privacidad!
  • Al hacer clic directamente en un resultado de búsqueda, usted ya no cuenta con la protección de Ixquick.
  • Se establece una conexión directa con el sitio web de un tercero.
  • Este sitio web puede ver y registrar su dirección de IP, colocar cookies en su navegador y, posiblemente, leer las cookies que otros colocaron allí.
  • El sitio web puede observar y registrar su comportamiento, incluidos los enlaces en los que hace clic y las páginas que ve.
  • El beneficio es que las páginas se cargan con mayor rapidez ya que el contenido se muestra directamente desde su fuente.
  • La opción “sin proxy” es conveniente cuando no le preocupa su privacidad, cuando confía en el sitio web o cuando desea ser identificado, como cuando ingresa a su cuenta bancaria.
Protección completa del ProxyHaga clic en “proxy” (mayor privacidad)

Ixquick Web Search
www.ixquick.comProxyDestacar¡Ixquick.com ofrece resultados de búsqueda precisos con total privacidad!
  • Ixquick se dirige al sitio web que usted seleccione, encuentra la página y se la muestra.
  • Usted es invisible para el sitio web. Sólo ven la dirección IP de Ixquick, no la suya.
  • Como nunca realiza un contacto directo con el sitio web, no pueden ver o almacenar cookies en su navegador.
  • Puede hacer clic en las páginas vinculadas desde el sitio web y el servidor proxy de Ixquick también las mostrará.
  • Mientras utilice el proxy, estará protegido en todo momento por la excelente política de privacidad de Ixquick. (No se registra la dirección de IP, no se utilizan cookies de identificación, no se almacena la actividad de búsqueda o navegación).
  • El Servicio Proxy de Ixquick brinda total anonimato y privacidad.
  • La desventaja es que las páginas tardan más en cargarse, ya que Ixquick debe obtener los contenidos y volver a mostrarlos.
Otros comentarios y restricciones:
  • Por motivos de seguridad, no puede utilizar el proxy para completar formularios con entrada de texto.
  • El proxy quita el código “Javascript” de los sitios visitados, por lo que es posible que algunas características no funcionen.
  • El proxy no cargará marcos que se encuentren en un dominio externo distinto al de referencia.
  • El servicio proxy actualmente no se amplía a búsquedas de video o imagen completa.
  • Para una protección más exhaustiva, puede inhabilitar las cadenas de ixquick-proxy.com con el complemento "NoScript" de Firefox.

Ixquick Proxy evita la mayoría de los problemas de seguridad al eliminar casi todo el código Javascript de las páginas procesadas como proxy.

Para mayor seguridad, Ixquick recomienda deshabilitar JavaScript, Java, Flash y otras secuencias de comandos de todo el dominio ixquick-proxy.com y sus subdominios.

Para hacerlo, siga los siguientes pasos:
1. Asegúrese de utilizar el explorador web Firefox
2. Instale el complemento NoScript

3. Para que sea más fácil utilizarlo, configure NoScript para:
  A. Permitir secuencias de comandos a nivel global, y
  B. Deshabilitar secuencias de comandos de ixquick-proxy.com y sus subdominios.

O si desea protección incluso cuando no está utilizando Starpage Proxy, en caso de algún inconveniente, configure NoScript para:
  A. Bloquear secuencias de comandos a nivel global, y
  B. Habilitar secuencias de comandos de ixquick.com, sus subdominios y cualquier otro sitio web de confianza, pero no para ixquick-proxy.com.
