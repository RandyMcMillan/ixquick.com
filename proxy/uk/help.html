Full Proxy ProtectionClick "Proxy" (more private)

Ixquick Web Search
www.ixquick.comProxyHighlightixquick.com provides accurate search results with full privacy!
  • Ixquick goes to the website you select, retrieves the page, and displays it for you.
  • You are invisible to the website. They see only Ixquick's IP address, not yours.
  • Since you never make direct contact with the website, they can't see or store cookies on your browser.
  • You can click on linked pages from the website and the Ixquick proxy will display those for you too.
  • While using the proxy, you are protected at all times by Ixquick's outstanding privacy policy. (No IP address recorded, no identifying cookies used, no search or browsing activity stored.)
  • The Ixquick Proxy Service offers complete privacy and anonymity.
  • The downside is that pages load more slowly, since Ixquick must retrieve the contents and redisplay them.
Additional Notes and Restrictions:
  • For security reasons, you cannot use the proxy to fill out forms with text input.
  • The proxy removes the "javascript" code of visited sites, so some features may not work.
  • The proxy will not load frames that reside on a different external domain than the referrer.
  • The proxy service currently doesn't extend to full image or video searches yet.
  • For even further protection, you can disable scripts from ixquick-proxy.com with the Firefox "NoScript" plugin.
    More


The Ixquick Proxy prevents most security problems by removing most Javascript code from the proxied pages.

For added security Ixquick recommends disabling Javascript, Java, Flash and other scripting from the entire ixquick-proxy.com domain and its subdomains.

To do so:
1. Make sure you are using the Firefox Web Browser
2. Install the NoScript Add-on

3. For greatest ease-of-use, configure NoScript to:
  A. Allow Scripts Globally, and
  B. Disable Scripts from ixquick-proxy.com and its subdomains.

Or, for protection even when you are not using the Ixquick Proxy, with some inconvenience, configure NoScript to:
  A. Block scripts globally, and
  B. Enable Scripts from ixquick.com, its subdomains, and any other trusted websites -- but not ixquick-proxy.com.
