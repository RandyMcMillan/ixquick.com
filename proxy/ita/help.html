Ixquick Web Search
www.ixquick.com ProxyEvidenziaixquick.com fornisce risultati di ricerca accurati con privacy totale!
  • Quando si clicca direttamente su un risultato di ricerca, si esce dalla protezione Ixquick.
  • Viene stabilita una connessione diretta con un sito web esterno.
  • Il sito web può vedere e registrare l’ indirizzo IP dell’utente, mettere cookie sul browser dell’utente e in alcuni casi leggerei cookie lasciativi da altri.
  • Il sito web è in grado di osservare e registrare il comportamento online dell’utente, inclusi i collegamenti cliccati e le pagine visualizzate.
  • L’aspetto positivo è che le pagine vengono caricate più rapidamente,dato che si visualizza un contenuto direttamente dalla fonte.
  • L’opzione "no-proxy" può essere valida se non si hanno problema di privacy, quando si ha fiducia in un certo sito web o quando si vuole essere identificati, come per esempio quando si accede al proprio conto bancario.
Protezione Proxy totaleCliccare su "Proxy" (maggiore privacy)

Ixquick Web Search
www.ixquick.comProxyEvidenziaixquick.com fornisce risultati di ricerca accurati con privacy totale!
  • Ixquick si porta sul sito web selezionato, trova la pagina e la visualizza.
  • L’utente è invisibile al sito web. Il sito “vede” solo l’indirizzo IP di Ixquick, non quello dell’utente.
  • Dato che l’utente non ha mai contatto diretto con il sito web, questi non può vedere o salvare i cookie sul browser dell'utente.
  • Si può cliccare sulle pagine di collegamento dal sito web e il Proxy Ixquick le apre.
  • Mentre si usa il proxy, si è protetti in qualsiasi momento grazie alla superiore privacy policydi Ixquick. (Non vengono registrati o identificati nessun indirizzo IP, cookie, attività di ricerca o di uso del browser).
  • Il servizio Ixquick Proxy offre privacy e anonimità assolute.
  • L’aspetto meno positivo è che le pagine vengono caricate più lentamente, dato che Ixquick deve trovare il contenuto e visualizzarlo nuovamente.
Annotazioni e limitazioni aggiuntive:
  • Per ragioni di sicurezza, non è possibile usare il proxy per compilare moduli con immissione di testo.
  • Il proxy rimuove il codice "javascript" dei siti visitati, conseguentemente alcune caratteristiche potrebbero non funzionare.
  • Il proxy non carica i frame che risiedono su un dominio esterno diverso da quello di riferimento.
  • Al momento il servizio proxy non comprende ancora ricerche di immagini o filmati.
  • Per una protezione ancora maggiore è possibile disattivare gli script in ixquick-proxy.com con il plugin di Firefox "NoScript".

Ixquick Proxy previene la maggior parte dei problemi di sicurezza rimuovendo la maggior parte del codice Javascript dalle pagine visitate via proxy.

Per ulteriore sicurezza Ixquick suggerisce di disattivare Javascript, Java e Flash e altri script dall'intero dominio ixquick-proxy.com e dai relativi sottodomini.

Per farlo:
1. Assicurarsi di usare il browser per il web Firefox
2. Installare l’estensione NoScript Add-on

3. Per una migliore facilità d’uso configurare NoScript come segue:
  A. Permetti script globalmente, e
  B. Disabilitare gli script da ixquick-proxy.com e dai suoi sottodomini.

Oppure, per una tutela anche quando non si sta usando Ixquick Proxy, sia pure con qualche piccolo inconveniente, configurare NoScript come segue:
  A. Blocca script globalmente, e
  B. Attiva Script da ixquick.com, dai suoi sottodomini e da altri siti web di fiducia -- ma non da ixquick-proxy.com.
