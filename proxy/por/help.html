Proteção total do proxyClique em "proxy" (mais privacidade)

Ixquick Web Search
www.ixquick.comProxyDestaqueO ixquick.com oferece resultados de pesquisa precisos com total privacidade!
  • O Ixquick vai até o site que você seleciona, recupera a página e a exibe para você.
  • Você fica invisível para o site. Ele vê apenas o endereço IP do Ixquick, e não o seu.
  • Como você nunca faz contato direto com o site, ele não é capaz de ver nem armazenar cookies no seu navegador.
  • Quando você clica em páginas com links do site, o Ixquick Proxy as exibe também.
  • Ao usar o proxy, você fica protegido ininterruptamente pela excelente política de privacidade do Ixquick. (Nada de registro do endereço IP, nada de uso de cookies de identificação, nada de armazenamento de atividades de pesquisa ou navegação.)
  • O Ixquick Proxy Service oferece privacidade e anonimato totais.
  • A desvantagem é que as páginas levam mais tempo para carregar, pois o Ixquick precisa recuperar o conteúdo e reexibi-lo.
Observações e restrições adicionais:
  • Por motivos de segurança, não é possível usar o proxy para preencher formulários com digitação de texto.
  • Como o proxy remove o código "javascript" dos sites visitados, alguns recursos podem não funcionar.
  • O proxy não carrega frames contidos em um domínio externo diferente daquele que fez o encaminhamento.
  • O serviço de proxy atualmente ainda não abrange pesquisas completas com imagem ou vídeo.
  • Para aumentar ainda mais a proteção, você pode desativar scripts de ixquick-proxy.com com o plugin “NoScript” do Firefox.
    Mais


O Ixquick Proxy evita a maioria dos problemas de segurança porque remove a maior parte do código Javascript das páginas com proxy.

Para aumentar a segurança, o Ixquick recomenda desativar o Javascript, Java, Flash e outros scripts do domínio ixquick-proxy.com inteiro e de seus subdomínios.

Para fazer isso:
1. Certifique-se de estar usando o navegador Firefox Web
2. Instale o complemento NoScript

3. Para facilitar o uso ao máximo, configure o NoScript para:
  A. Permitir scripts no nível global e
  B. Desative os scripts do domínio ixquick-proxy.com e seus subdomínios.

Para ter proteção mesmo quando não estiver usando o Ixquick Proxy, mas com certo incômodo, você também pode configurar o NoScript para:
  A. Bloquear scripts no nível global e
  B. Ativar scripts em ixquick.com, seus subdomínios e qualquer outro site confiável – mas não no ixquick-proxy.com.
