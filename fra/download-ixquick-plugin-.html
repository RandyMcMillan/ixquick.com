    Ajouter au navigateur

    Vous pouvez dès à présent ajouter facilement Ixquick à votre liste de moteurs de recherche de Firefox. Les moteurs de recherche apparaissent dans le menu déroulant à côté de la barre de recherche dans votre navigateur Firefox. Les réglages que vous avez enregistrés dans "Préférences" seront conservés.

    Pour ajouter Ixquick comme moteur de recherche dans Edge, procédez comme suit:

    1. Cliquez sur l'icône à trois points en haut à droite:

    2. Dans le menu qui apparaît, sélectionnez Paramètres.
    3. Faites défiler la liste, et sous paramètres avancés, choisissez Voir les paramètres avancés.
    4. Faites défiler vers le bas, et dans le menu déroulant Rechercher dans la barre d'adresse avec, sélectionnez <Ajouter nouveau≷.
    5. Vous voyez apparaître une liste de moteurs de recherche. Sélectionnez Ixquick.

    Vous pouvez aussi faire de Ixquick votre moteur de recherche favori, dans le navigateur Safari. Pour ce faire, veuillez installer notre extension en suivant ces étapes. Cela ne modifiera aucun réglage de votre navigateur.


    1. Dans votre navigateur Safari, validez les  extensions. Pour cela, cliquez sur le mot 'Safari' dans la barre de menu de Mac OS (près de l'icône Apple). Dans le menu qui apparaît, choisissez 'Préférences', alors 'Extensions', et placez le commutateur sur " Activé ". Vous pouvez maintenant  refermer  la fenêtre des préférences.
    2. Cliquez sur le lien ci-dessous, pour télécharger l'  extension :
      Extension 'Ixquick' pour Safari
    3. Dans le menu Safari des téléchargements, faites un double-clic  sur le fichier 'Ixquick-HTTPS.safariextz', et si nécessaire, cliquez sur  'Ouvrir'.
    4. Safari vous demande si vous voulez procéder à l'installation. Pour parachever l'action, cliquez sur le bouton 'Installer'.

    Vous êtes désormais prêt à effectuer des recherches avec Ixquick, à partir de la barre de recherches de Safari. Bonnes recherches !

    Vous pouvez ajouter Ixquick à la boîte de recherche Safari par le biais de l'addiciel "Glims". Les réglages que vous avez enregistrés dans "Préférences" seront conservés.
    Instructions:

    1. Veuillez télécharger et installer l'addiciel "Glims" à partir de la page : http://www.machangout.com/. Redémarrez ensuite Safari
    2. Allez dans Safari -> Préférences et cliquez sur l'icône "Glims"
    3. Sur la section gauche, cliquez sur "Moteurs de recherche"
    4. Veuillez cliquer sur le bouton "Ajouter" situé en bas de page
    5. Dans la fenêtre qui s'ouvre,
         a) tapez lxquick dans le champ "Nom"
         b) in "Query URL" field, replace the https://www.google.com/q=#query# with the
             https://ixquick.com/do/metasearch.pl?language=francais&query=#query#
            (assurez-vous que l'URL contienne '&query=#query#')
    6. Cliquez sur le bouton "Ajouter" et faites défiler la liste vers le bas jusqu'à ce que vous trouviez Ixquick. Faites-le alors glisser vers le haut avec votre souris (vous pourriez également choisir de ne cocher aucun des autres moteurs de recherche)
    7. Fermez Safari -> Préférences et redémarrez Safari. Ixquick devrait maintenant être visible dans votre zone de recherche. Tapez un terme de recherche et [Entrée]

    Vous pouvez maintenant facilement ajouter Ixquick à votre navigateur Safari.
    Le moteur de recherche Ixquick apparaît alors dans le menu déroulant à côté de la barre de recherches dans votre navigateur.
    Instructions:

    1. Cliquez sur le menu déroulant de la boîte de recherche de votre navigateur.
    2. Sélectionnez «Préférences AcidSearch… ou Édition des canaux de recherche…» à partir du menu déroulant.
    3. Téléchargez et installez le module d’extension Acidsearch à l’adresse http://www.macupdate.com/info.php/id/15495 si vous n’en disposez pas.
    4. Après avoir installé le module d’extension AcidSearch et relancé Safari, réitérez les étapes 1 et 2, et passez à l’étape 5.
    5. Cliquez sur un des canaux de recherche pour activer les options et cliquez sur « Nouveau canal » (options en bas à droite).
    6. Saisissez « Ixquick » dans le champ « Titre ».
    7. Dans « Préfixe URL », saisissez https://startpage.com/do/metasearch.pl?query=$0 s’il s’agit d’AcidSearch 0.7b3 ou de versions plus récentes. Dans le cas contraire, utilisez cette adresse url: https://startpage.com/do/metasearch.pl?query=
    8. Fermez la fenêtre et choisissez Ixquick dans la boîte de recherche.

    Vous pouvez maintenant facilement ajouter Ixquick à votre navigateur Safari Konqueror.
    Pour ajouter un raccourci Internet vers Ixquick à votre navigateur Konqueror, suivez les instructions ci-dessous.
    Instructions:

    1. Allez à « Settings » (configuration) dans le menu Konqueror de votre navigateur.
    2. Sélectionnez « configurez konqueror ».
    3. Choisissez « Enhanced Browsing or Web Shortcuts » (navigation améliorée ou raccourcis Internet) dans la barre latérale de gauche, et une longue liste de raccourcis s’affiche.
    4. Cliquez sur le bouton « New » (Nouveau).
    5. Saisissez « Ixquick » dans le nom du prestataire de services de recherche.
    6. Saisissez 'https://ixquick.com/do/metasearch.pl?query=\{@}' dans l’adresse de recherche :
    7. Saisissez vos propres raccourcis, par exemple : « ix,ixq,ixquick » dans les raccourcis URI :
    8. Cliquez sur OK.


    Lorsque vous avez suivi ces instructions, vous êtes prêt à effectuer une recherche : il vous suffit de saisir « ix: privacy » et d’appuyer sur Enter !
