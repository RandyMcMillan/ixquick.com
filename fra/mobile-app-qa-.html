    Ixquick Search pour iOS et Android est arrivé !



    L'application Ixquick Search pour iPhone, iPod Touch, iPad et tous les appareils Android vous offre toute la confidentialité à laquelle vous vous attendez de la part d'Ixquick — en plus de certaines autres caractéristiques spéciales pour protéger votre navigation mobile.

    Le navigateur intégré de Ixquick Search rejette tous les témoins, empêche les sites que vous consultez de voir vos termes de recherche et truque même votre agent d'utilisateur de sorte que les sites Web ne peuvent pas l'utiliser pour vous identifier.

    En outre, Ixquick Search efface vos recherches et les pages Web après un certain délai... ou lorsque vous cliquez sur le bouton Home (accueil). Vous serez protégé contre des tiers qui voudraient s'approprier physiquement votre appareil et fouiller votre historique de recherche.

    Installez-la directement à partir d'un appareil Android

    1. Activez votre appareil pour permettre l'installation d'applications qui ne proviennent pas de la boutique Google Play. Dans l'ancienne version Android 4.3, cette option se trouve dans Paramètres -> Sécurité, où la case « Sources inconnues » doit être cochée. Les mesures exactes à prendre peuvent varier d'un appareil à l'autre, donc consultez la documentation du fabricant ou le site de soutien de votre appareil Android pour effectuer cette action, au besoin.

    2. Téléchargez et installez l'application StartPage Private Search en téléchargeant le fichier Android Application Package (.apk) suivant. Pour obtenir des instructions particulières sur le téléchargement de logiciels, reportez-vous à la documentation de votre appareil mobile. Toutefois, le processus devrait se lancer si vous cliquez sur le fichier StartPage dans la section Téléchargements.

    Q: Qu'est-ce qui rend l'application mobile de Ixquick si confidentielle?


    L'application Ixquick Search comporte les mêmes caractéristiques de confidentialité qui font de Ixquick le moteur de recherche le plus confidentiel du monde.
    • Toutes les recherches sont cryptées.
    • Nous n'enregistrons ni votre visite ni vos recherches.
    • Nous ne divulguons jamais aux sites ce que vous avez recherché.
    • Le Ixquick Proxy vous rend invisible sur les sites Web que vous consultez.
    En plus de cela, nous avons ajouté des fonctionnalités qui protègent votre vie privée lorsque vous effectuez des recherches et naviguez sur votre appareil mobile :
    • Tous les témoins (cookies) de suivi sont rejetés.
    • Nous fournissons aux sites Web un faux « agent d'utilisateur » pour les empêcher d'établir un profil de votre appareil mobile
    • Nous supprimons les recherches et les pages Web au fur et à mesure ou lorsque vous suspendez l'application, de sorte que personne ne peut les voir.

    Q: J'ai cliqué sur la flèche Précédent pour retourner aux résultats ou aux pages Web que j'avais consultés, mais ils ont disparu maintenant. Pourquoi?

    L'application Ixquick Search a été conçue pour protéger votre vie privée contre tout examen physique indésirable de votre appareil mobile par des tiers.

    Vos résultats et vos pages Web restent visibles pendant une certaine période de temps pour rendre la navigation plus facile, mais ensuite ils disparaissent de sorte que personne d'autre ne peut les voir. Lorsque cela se produit, vous verrez un message vous informant que la page a été supprimée.

    Vous pouvez établir la période de temps à 5, 15 ou 30 minutes, ou désactiver cette fonctionnalité dans les paramètres (la roue dentée du coin inférieur droit).

    Q: Je viens de suspendre l'application, puis j'y suis revenu. Pourquoi mes résultats et pages Web ont-ils disparu?

    Lorsque vous quittez l'application Ixquick Search, vos résultats et pages Web disparaissent de sorte que personne d'autre ne peut les voir - tout comme ils disparaissent après un certain délai d'inactivité. Sur Ixquick, le simple fait d'appuyer sur le bouton Home (accueil) protège votre vie privée contre des intrus physiques.

    Tout comme dans le cas du délai d'inactivité, vous verrez un message vous informant que la page que vous consultiez a été supprimée. Tout comme vous le feriez pour le délai d'inactivité, vous pouvez désactiver cette fonctionnalité dans les paramètres (roue dentée du coin inférieur droit).

    Q: Pourquoi l'application fonctionne-t-elle plus lentement quand j'utilise le proxy?

    Tout comme le proxy de startpage.com, celui de l'application Ixquick Search copie chaque page Web que vous consultez sur nos propres serveurs protégés sur le plan de la confidentialité, de sorte que vous la voyez à travers nous, pas eux. Cela vous donne un maximum de confidentialité parce que vous n'avez jamais de contact direct avec la page Web.

    Étant donné qu'il faut du temps pour copier chaque page sur notre serveur et ensuite vous la faire afficher, le proxy navigue sur le Web plus lentement.

    Q: Pourquoi certains sites Web semblent-ils différents à travers le proxy?

    Certains sites Web incluent des versions mobiles qui sont optimisées pour les petits écrans. Toutefois, lorsque vous utilisez la fonction de protection de la vie privée du proxy, ces sites Web ne peuvent pas déterminer que vous utilisez un appareil mobile. Ainsi, vous verrez la version normale de bureau à travers le proxy.

    Le proxy bloque également Javascript, un élément de certaines pages Web qui peut transmettre des renseignements personnels. Le blocage de Javascript peut donner à certains sites Web un aspect différent.

    Q: Puis-je changer la façon dont les résultats s'affichent?

    Sur la page Paramètres (cliquez sur la roue dentée du côté droit de la barre verte de bas de page), vous pouvez choisir de rendre le texte des résultats plus ou moins grand et vous pouvez choisir d'afficher 10, 20 ou 50 résultats par page.

    Q: Puis-je filtrer les résultats par date?

    Vous pouvez limiter les résultats du Web aux dernières 24 heures, à la dernière semaine, au dernier mois ou à la dernière année à l'aide de la liste déroulante qui apparaît dans le coin supérieur droit de la page des résultats

    Q: Quand je navigue sur le Web à l'aide de Ixquick, comment puis-je effectuer une nouvelle recherche?

    Il suffit de cliquer sur la barre de recherche du haut de votre écran et d'effectuer votre nouvelle recherche. Vous pouvez utiliser le "x" gris situé dans un cercle pour supprimer le texte de votre recherche précédente.

    Q: L'application me pose un problème. Comment puis-je vous demander de l'aide?

    Écrivez-nous un e-mail à app@ixquick.com en décrivant le problème, ainsi que ce que vous faisiez quand cela s'est produit. Bien sûr, vous ne devez pas nous divulguer le contenu de vos recherches confidentielles !
