    使用進階語法設計最佳的搜尋!

    您可以使用進階語法讓搜尋更加有效率。範例如下:

    All of the terms:
    UFO AND abduction    		Place the word AND between two words or phrases that must be included. Ixquick will only return pages that contain all those words.

    If some words are mandatory but others are optional, place the + before only the mandatory words (for example, green +jello +pudding if you are less concerned about the color but are aching for a tasty jello pudding).
    省略一個字串：
    貓-凱蒂    		 在一定不可出現的字詞前面擺一個減號(-)。或是把「排除」（NOT）、「也排除」（ANDNOT）等字眼擺在前面。

    例如，如果你要尋找貓的網站，可是不想包括凱蒂貓，就請輸入貓-凱蒂。
    片語：
    "南方公園"    		 如果所有關鍵字必須連在一起組成一個片語，而非隨意散佈在網頁中，就把它們擺進引號中。例如，"南方公園 "的搜尋比較可能找到談論暴力電視節目「南方公園」的網站，而非座落在法國南部的公園。
    括弧：
    領養以及 (小獵犬或是吉娃娃)
    領養AND (小獵犬OR吉娃娃)     		在括號內混合「以及」（AND）、「或是」(OR)、「不包括」(NOT)，以便建立複雜的搜尋。搜尋領養以及(小獵犬或吉娃娃)，就會找到包含領養與小獵犬，領養與吉娃娃，或這三個字一網打盡的網頁。
    欄位： 指定哪些資訊必須顯示欄位。目前支援的欄位包括:
    +標題:"丹奎爾 " + 門撒國際標題: 網頁標題必須包含搜尋的文字。
    +網域:英國 + 披頭四網域: 網頁必須位於 .com 、 .gov 、 .edu 或 .uk (等) 的電腦上
    主機:nytimes.com主機: 網頁必須在此特定電腦或更多特定的電腦上 (例如主機:nytimes.com 會擷取在nytimes.com 及 www.nytimes.com) 上的網頁
    圖片:雲圖片: 網頁必須包含圖片，此圖片的檔案名稱包含搜尋的文字。
    url:bigmanurl: 在這個 url 下找出所有包含指定文字的網頁，例如 url:uk 就會找到 http://abc.com/pics/uk.html 以及 http://www.uk/
    連結:nytimes連結: 網頁必須包含一個超連結，其目的地位址的開端必須包含至少一個指定文字。
