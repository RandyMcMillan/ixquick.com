    Ixquick Search för iOS och Android är här!



    Ixquick Search för iPhone, iPod Touch, iPad och alla Android-enheter ger dig all den sekretess som du förväntar dig från Ixquick – samt ett par ytterligare specialfunktioner för att skydda din mobila surfning.

    Den inbyggda webbläsaren i Ixquick Search avvisar alla spårningscookies, ser till att de besökta webbplatserna inte kan se dina sökord och skickar till och med en falsk användaragent för att webbplatserna inte ska kunna använda den för att identifiera dig

    Dessutom raderar Ixquick Search dina sökningar och webbsidor efter en viss tidsgräns... eller när du knackar på hemknappen. På så sätt skyddas du mot att tredje parter tar din enhet fysiskt och undersöker din sökhistorik.

    Klicka här för att hämta Ixquick Search för iPhone från App Store.




    Klicka här för att hämta Ixquick Search för Android från Google Play.

    Klicka här för att hämta Ixquick Search för Android från Amazon App Store.

    Android-användare kan även hämta Ixquick Private Search direkt från oss utan att använda Google Play.

    Installera direkt på en Android-enhet:

    1. Gör så att applikationer som inte kommer från Google Play kan installeras på din enhet. I standardversionen 4.3 av Android finns detta alternativ i Settings (Inställningar) -> Security (Säkerhet). "Unknown Sources" (Okända källor) måste vara markerat. De specifika stegen kan variera från en enhet till en annan. Kontrollera tillverkarens dokumentation eller supportwebbplats för din specifika Android-enhet för att utföra detta steg om det behövs.

    2. Hämta och installera applikationen Ixquick Private Search genom att hämta följande Android Application Package-fil (.apk). Var god se dokumentationen för din mobila enhet för specifika instruktioner om hur du hämtar programvara. En knackning på Ixquick-filen bland dina hämtade filer bör dock starta processen.

    FRÅGA: Vad är det som gör Ixquick mobilapp så privat?


    Ixquick Search inkluderar samma sekretessfunktioner som gör Ixquick till världens mest privata sökmotor.
    • Alla sökningar krypteras.
    • Vi registrerar inte ditt besök eller dina sökningar.
    • Vi berättar aldrig för webbplatser vad du sökte.
    • Ixquick proxytjänst gör dig osynlig för de webbplatser som du besöker.
    Dessutom har vi lagt till funktioner som skyddar din personliga information medan du söker och surfar på din mobila enhet:
    • Alla spårningscookies avvisas.
    • Vi förser webbplatser med en falsk "användaragent" för att förhindra att de samlar information och skapar en profil för din mobila enhet
    • Vi raderar sökningar och webbsidor efter att en viss tid har gått och när du lämnar applikationen, så att ingen annan kan se dem.

    FRÅGA: Jag klickade på bakåtpilen för att gå tillbaka till resultat eller webbsidor som jag hade tittat på och nu är de borta. Varför?

    Ixquick Search är designad för att skydda din personliga information mot oönskad fysisk granskning av din mobila enhet från tredje parter.

    Dina resultat och webbsidor förblir synliga under en viss tid för att underlätta din surfning på nätet men sedan försvinner de så att ingen annan kan se dem. När det sker ser du ett meddelande om att sidan har tagits bort.

    Du kan sätta tidsperioden på 5, 15 eller 30 minuter eller inaktivera den här funktionen i Settings (Inställningar) (kugghjulet i nedre högra hörnet).

    FRÅGA: Jag lämnade applikationen och gick sedan tillbaka. Varför är mina resultat och webbsidor borta?

    När du lämnar Ixquick Search försvinner dina resultat och webbsidor så att ingen annan kan se dem – precis som de gör efter en viss tidsfördröjning. Med Ixquick räcker det att trycka på hemknappen för att skydda din personliga information mot eventuella fysiska intrång.

    Precis som med tidsfördröjningen får du se ett meddelande om att sidan som du tittade på har tagits bort. Dessutom kan du, precis som med tidsfördröjningen, stänga av den här funktionen i Settings (Inställningar). (kugghjulet i nedre högra hörnet)

    FRÅGA: Varför är appen långsammare när jag använder proxyn?

    Både proxyn för StartPage Search och proxyn för startpage.com kopierar varje webbsida som du besöker till våra egna sekretesskyddade servrar och du ser därför webbsidan genom oss, inte genom dem. Det ger dig ett ultimat skydd av din personliga information eftersom du aldrig har direkt kontakt med webbsidan.

    Eftersom det tar längre tid att kopiera varje sida till vår server och sedan visa den för dig är det långsammare att surfa på nätet med proxyn.

    FRÅGA: Varför ser vissa webbplatser annorlunda ut genom proxyn?

    En del webbplatser har mobilversioner som är optimerade för små skärmar. Men när du använder proxyns sekretesskydd märker webbplatserna inte ens att du använder en mobil enhet! Proxyn visar därför den normala skrivbordsversionen.

    Proxyn blockerar också JavaScript, som används på vissa webbsidor och kan överföra personlig information. Blockeringen av JavaScript kan göra att vissa webbplatser ser annorlunda ut.

    FRÅGA: Kan jag ändra hur resultaten ser ut?

    På sidan Settings (Inställningar) (knacka på kugghjulet till höger i den gröna sidfoten) kan du göra texten i resultaten större eller mindre och välja att visa 10, 20 eller 50 resultat per sida.

    FRÅGA: Kan jag filtrera resultaten efter datum?

    Du kan begränsa webbresultaten till de senaste 24 timmarna, den senaste veckan, den senaste månaden eller det senaste året genom att använda menyn i det övre, högra hörnet på resultatsidan.

    FRÅGA: Hur gör jag en ny sökning när jag surfar på nätet med Ixquick?

    Knacka på sökfältet överst på skärmen och ange din nya söksträng. Du kan knacka på krysset i den grå cirkeln för att ta bort din tidigare söksträng.

    FRÅGA: Jag har ett problem med appen. Hur kan jag kontakta er för att få hjälp?

    Skriv ett e-postmeddelande till oss på app@ixquick.com med en beskrivning av problemet och vad du höll på med när det inträffade. Självklart behöver du inte berätta innehållet i dina privata sökningar för oss!
