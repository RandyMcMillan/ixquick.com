Il generatore di URL

Salvare le impostazioni senza cookie

  • Per salvare le preferenze su Ixquick fare clic sul pulsante "Salva Impostazione" nella pagina "Impostazioni".
  • Così facendo si salveranno le impostazioni sul computer, in un 'cookie', un piccolo file di dati salvato sul disco rigido, chiamato “preferenze”
  • Il cookie “preferenze” di Ixquick è un cookie non identificabile che può essere accettato in sicurezza. In generale tuttavia i cookie possono rappresentare una minaccia per la privacy.
  • Chi preferisce non accettare cookie o cancellarli con regolarità non potrà salvare le preferenze in questo modo.
  • Per questi casi Ixquick ha creato appositamente il pulsante "Genera URL" in fondo alla pagina "Impostazioni".
  • Cliccando sul pulsante viene generato un URL specifico di Ixquick contenente le impostazioni selezionate.
  • L’URL può essere generato in formato leggibile od offuscato.
  • L’URL generato può ora essere impostato direttamente quale homepage o copiato e usato quale preferito.
  • Utilizzando questo URL tutte le preferenze saranno utilizzate alla successiva visita su Ixquick.
  • Per maggiore praticità, è possibile selezionare "Installa plugin personalizzato" dopo aver creato l’URL. Sarà così possibile installare un plugin per la barra di ricerca contenente le proprie impostazioni.
  • Può essere necessario eliminare prima le versioni precedentemente salvate dall’elenco a discesa della barra di ricerca.
  • Verificare che la parole chiave immessa sia diversa dalla parola chiave precedentemente salvata.
Ixquick continua a offrire funzionalità a tutela della privacy per effettuare ricerche in rete in sicurezza!
