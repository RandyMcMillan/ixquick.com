    Dem Browser hinzufügen

    Sie können Ixquick jetzt Ihrer Liste der Firefox-Suchmaschinen hinzufügen. Die Suchmaschinen erscheinen bei Ihrem Firefox-Browser durch das Anklicken des kleinen Pfeils neben dem Suchfeld. Dabei werden die Einstellungen – wie auf Ixquick festgelegt – übernommen.

    Sie können Ixquick jetzt Ihrer Liste der Iceweasel-Suchmaschinen hinzufügen. Die Suchmaschinen erscheinen bei Ihrem Iceweasel-Browser durch das Anklicken des kleinen Pfeils neben dem Suchfeld. Dabei werden die Einstellungen – wie auf Ixquick festgelegt – übernommen.

    Sie können Ixquick jetzt Ihrer Liste der Icecat-Suchmaschinen hinzufügen. Die Suchmaschinen erscheinen bei Ihrem Icecat-Browser durch das Anklicken des kleinen Pfeils neben dem Suchfeld. Dabei werden die Einstellungen – wie auf Ixquick festgelegt – übernommen.

    Leider unterstützt Ihr Browser keine Suchmaschinen-Plugins.Das Suchmaschinen-Plugin funktioniert mit dem Internet Explorer 7.0+ (und höher) sowie Firefox.

    Fügen Sie jetzt Ixquick der Leiste Ihrer Suchmaschinen im Internet Explorer 7+ (und höher) zu. Die Suchmaschinen sind im Internet Explorer 7+ Browser neben dem Eingabefeld für Suchwörter zu finden. Dabei werden die Einstellungen – wie auf Ixquick festgelegt – übernommen.

    Fügen Sie jetzt Ixquick der Leiste Ihrer Suchmaschinen im Internet Explorer 7+ (und höher) zu. Bitte nutzen Sie die folgende Adresse von der offiziellen Internet Explorer Donwloadseite, um das PlugIn für IE 9+ herunterzuladen und zu installieren.

    http://www.iegallery.com/de-de/Addons/Details/11825 — Ixquick HTTPS - Deutsch

    Nach der Installation wählen Sie im Popup Menü der Adressleiste Ixquick aus.
    Schon sind Sie bereit, drauflos zu suchen: Einfach einen Suchbegriff eingeben und auf Enter drücken.

    2. Wählen Sie aus dem auftauchenden Menü den Menüpunkt Einstellungen.

    Sie können Ixquick auf Safari auch als Standardsuchmaschine nutzen. Wenn Sie das möchten, folgen Sie bitte den Instruktionen unterhalb. Durch die Installation werden keine anderen Dienste oder Programme beeinflusst.


    1. Um die Erweiterungen in Safari zu aktivieren starten sie bitte Safari. Öffnen Sie nun die Einstellungen und klicken Sie auf die Lasche „Erweiterungen“. Dort stellen Sie den Ein/Aus Button auf „Ein“. Schließen Sie nun die Einstellungen.

      Ixquick Safari Erweiterung
    3. Doppel-Klick auf die Datei 'Ixquick-HTTPS.safariextz' im Safari Download-Menü und danach auf „Öffnen“
    4. Es öffnet sich nun Safari und fragt, ob die Erweiterung installiert werden soll. Klicken Sie auf „Installieren“ um den Prozess abzuschließen.

    Sie können nun die Ixquick Suche in Ihrem Safari Browser nutzen. Genießen Sie es.

    Ixquick zu Opera hinzufügen

    Sie können eine Suchmaschine bei Opera hinzufügen, indem Sie auf Einstellungen und dann auf den Button „Suchmaschinen verwalten“ klicken und dann auf „Eine neue Suche hinzufügen“. Dort geben Sie bitte die folgenden Details ein:

    Name: Ixquick
    Schlüsselbegriff: ix <Oder ein anderes Suchwort Ihrer Wahl>
    URL: https://ixquick.com/do/search?query=%s&cat=web&language=deutsch

    Dann klicken Sie bitte auf „fertig“.

    Sie können Ixquick nun nutzen, indem Sie Ihr Suchwort einfach in der Adressleiste des Browsers eingeben.

    Apple hat den Support für Safari auf dem Betriebssystem Windows eingestellt. Leider können wir Ihnen deshalb auch kein Browser Plug-In anbieten. allerdings bieten wir jetzt auch eine Browser Toolbar an, die über die selben Spezifikationen verfügt. Erfahren Sie hier mehr darüber.

    Derzeit gibt es keine Lösung für die Integration eines Plugins für Ixquick in Ihren Camino-Browser.

    Aufgrund bekannter Problemen bei Safari 3.2+ kann es bei der Installation von Plugins von Drittanbietern, wie z.B. AcidSearch, zu einem Absturz kommen.
    Apple arbeitet bereits an einer Lösung des Problems.
    Im Augenblick empfehlen wir daher, in Kombination mit Safari 3.2+. KEINE Plugins herunterzuladen.

    Sie können Ixquick im Suchfeld Safari über das Add-On "Glims" des Drittanbieters hinzufügen. Dabei werden die Einstellungen – wie auf Ixquick festgelegt – übernommen.
    Und so gehts:

    1. Laden Sie das Add-On "Glims" unter folgender Adresse herunter und installieren Sie es: http://www.machangout.com/ und starten Sie Safari neu
    2. Gehen Sie zu Safari -> Preferences und klicken Sie auf das Symbol "Glims"
    3. Klicken Sie im linken Abschnitt auf "Search Engines"
    4. Klicken Sie unten auf die Schaltfläche "Add"
    5. In dem nun angezeigten Feld,
         a) Geben Sie Folgendes in das Feld "Name" ein: Ixquick
         b) in "Query URL" field, replace the https://www.google.com/q=#query# with the
             https://ixquick.com/do/metasearch.pl?language=deutsch&query=#query#
            (stellen Sie sicher, dass die URL '&query=#query#' enthält)
    6. Klicken sie auf die Schaltfläche "Add" und scrollen Sie bis zum Eintrag Ixquick. Ziehen Sie den Eintrag mit der Maus nach oben (Sie können auch die Markierungen aller anderen Suchmaschinen aufheben)
    7. Schließen Sie die Safari -> Preferences und starten Sie Safari neu. Jetzt müsste Ixquick in Ihrem Suchfeld zu sehen sein. Geben Sie einen Suchbegriff ein und drücken Sie die [Enter]-Taste

    Jetzt können Sie Ixquick ganz einfach zu ihrem Safari Browser hinzufügen.
    Die Ixquick-Suchmaschine erscheint dann im Drop-Down-Menü neben der Suchleiste in ihrem Browser.
    Und so gehts:

    1. Klicken Sie auf den Dropdown-Pfeil im Suchfeld Ihres Browsers.
    2. Wählen Sie aus dem Drop-Down-Menü den Menüpunkt „AcidSearch Einstellungen ...“ oder „Suchkanäle bearbeiten ..“ aus.
    3. Wenn diese Option nicht vorhanden ist, laden Sie das Acidsearch-Plugin von http://www.macupdate.com/info.php/id/15495 herunter.
    4. Nach der Installation des AcidSearch-Plugins und dem Neustart von Safari wiederholen Sie die Schritte 1 und 2 und gehen dann zu Schritt 5.
    5. Klicken Sie auf einen der Suchkanäle, um die Optionen zu aktivieren und klicken Sie dann auf „Neuer Kanal“ (Optionen unten rechts).
    6. Als „Titel“ geben Sie „Ixquick“ ein.
    7. Als „Prefix URL“ geben Sie https://startpage.com/do/metasearch.pl?query=$0 ein. Wenn Sie AcidSearch 0.7b3 oder eine neuere Version benutzen, geben Sie diese URL ein. https://startpage.com/do/metasearch.pl?query=
    8. Schließen Sie das Fenster und wählen Sie Ixquick im Suchfeld.


    Wählen Sie nach der Installation im Dropdown-Menü des Suchfelds Ixquick aus.
    Nun können Sie mit der Suche beginnen: Geben Sie einfach den Suchbegriff ein und drücken Sie die Enter-Taste!

    Jetzt können Sie Ixquick ganz einfach zu ihrem Konqueror Browser hinzufügen.
    Die nachfolgende Anleitung beschreibt, wie Sie Ixquick als Web-Kurzbefehl zu Ihrem Konqueror Browser hinzufügen können.
    Und so gehts:

    1. Gehen Sie zum Menüpunkt „Einstellungen“ Ihres Browsers.
    2. Wählen Sie „Konqueror einrichten“.
    3. Wählen Sie aus der linken Leiste „Enhanced Browsing oder Web-Kurzbefehl“ aus, dann wird Ihnen eine schöne lange Liste von Kurzbefehlen angezeigt.
    4. Klicken Sie auf die Schaltfläche „Hinzufügen“
    5. Geben Sie „Ixquick“ als Name des Suchmaschinenanbieters an.
    6. Als Such-URI geben Sie ein: „https://ixquick.com/do/metasearch.pl?query=\{@}“
    7. Unter URI-Kurzbefehle können Sie ein leicht zu merkendes Kürzel angeben z.B: „ix“, „ixq“ oder „ixquick“
    8. Klicken Sie auf OK.
